Chain of Responsibility

 Posted on January 1, 0001  |  5 minutes  |  907 words  |  Kunlei Lian

Chain of Responsibility is one of the behavioral design patterns popularized by the original GoF book. It involves a sender object and potentially multiple receiver objects, and the point of this pattern is to decouple the sender from various receivers. Without this pattern, a sender would have to keep reference to all possible receivers and also define the logic of how the sender is processed by receivers.

A good example I can think of is the car maintenance service. Every once in a while, I would take my car for routine maintenance and after checking in at front desk, I don’t need to worry about how the car is serviced in the shop, the only ‘object’ I, as a customer, interacts with is the front desk. In this example, a customer is natually decoupled with maintenance mechanics.

Let’s first define a Car object:

 1/**
 2 * Car object to be serviced
 3 * @author Kunlei Lian
 4 */
 5public class Car {
 6    private boolean serviced;
 7    private boolean needAir;
 8    private boolean needOilChange;
 9    private boolean needNewBattery;
10    private boolean needNewWiper;
11
12    public Car() {
13        serviced = false;
14        needAir = true;
15        needOilChange = true;
16        needNewBattery = false;
17        needNewWiper = true;
18    }
19
20    public boolean isServiced() {
21        return serviced;
22    }
23
24    public void setServiced(boolean serviced) {
25        this.serviced = serviced;
26    }
27
28    public boolean needAir() {
29        return needAir;
30    }
31
32    public boolean needOilChange() {
33        return needOilChange;
34    }
35
36    public boolean needNewBattery() {
37        return needNewBattery;
38    }
39
40    public boolean needNewWiper() {
41        return needNewWiper;
42    }
43
44    @Override
45    public String toString() {
46        return "Car{" +
47                "needAir=" + needAir +
48                ", needOilChange=" + needOilChange +
49                ", needNewBattery=" + needNewBattery +
50                ", needNewWiper=" + needNewWiper +
51                '}';
52    }
53}

Next, we define a Service interface to be implemented by all possible receivers of a car.

1/**
2 * Service interface
3 * @author Kunlei Lian
4 */
5public interface Service {
6    void setNext(Service service);
7    void process(Car car);
8}

Then, we define various receivers:

 1/**
 2 * Front desk is the only receiver that interacts with a customer
 3 * @author Kunlei Lian
 4 */
 5public class FrontDesk implements Service {
 6    private Service nextService;
 7
 8    public FrontDesk() {
 9        nextService = null;
10    }
11
12    @Override
13    public void setNext(Service service) {
14        nextService = service;
15    }
16
17    @Override
18    public void process(Car car) {
19        if (car.isServiced()) {
20            System.out.println("FrontDestk: all necessary services performed, see you next time!");
21        } else {
22            System.out.println("FrontDesk: new car service request received, checking condition: ");
23            System.out.println("\t" + car.toString());
24            System.out.println("FrontDesk: sending car to service queue: ");
25
26            nextService.process(car);
27        }
28    }
29}
 1/**
 2 * TireService class adds air if needed
 3 * @author Kunlei Lian
 4 */
 5public class TireService implements Service {
 6    private Service nextService;
 7
 8    public TireService() {
 9        nextService = null;
10    }
11
12    @Override
13    public void setNext(Service service) {
14        nextService = service;
15    }
16
17    @Override
18    public void process(Car car) {
19        if (car.needAir()) {
20            System.out.println("TireService: Adding air to car!");
21        }
22        car.setServiced(true);
23        nextService.process(car);
24    }
25}
 1/**
 2 * OilService performs oil change if necessary

20            System.out.println("OilService: performing oil change!");

Last, we run a demo of the system:

 1/**
 2 * Car dealership with free services :)
 3 * @author Kunlei Lian
 4 */
 5public class ServiceCenter {
 6
 7    public static void main(String[] args) {
 8        Car car = new Car();
 9        FrontDesk frontDesk = new FrontDesk();
10        TireService tireService = new TireService();
11        OilService oilService = new OilService();
12        BatteryService batteryService = new BatteryService();
13        WiperService wiperService = new WiperService();
14
15        // setup chain of responsibility
16        frontDesk.setNext(tireService);
17        tireService.setNext(oilService);
18        oilService.setNext(batteryService);
19        batteryService.setNext(wiperService);
20        wiperService.setNext(frontDesk);
21
22        // start processing
23        frontDesk.process(car);
24    }
25}
