Visitor Pattern

   |  4   |  681   |  Kunlei Lian

Visitor Pattern is one of the behavioral patterns defined in the original GoF book. It is used to decouple computations from the objects they operate on. Say you have a group of classes that are related somehow and you have computations defined on top of them, instead of including the computation process inside the classes, visitor pattern lets you define separate computation classes that are unware of the internal representations of the classes they are going to work on.

This pattern needs to interfaces - Visitor and Visitable. Below is an example to compute the average weight and height of a family (assume size of 4).

1public interface Visitor {
2    void visit(Father father);
3    void visit(Mother mother);
4    void visit(Daughter daughter);
5    void visit(Son son);
6}
1public interface Visitable {
2    void accept(Visitor visitor);
3}

Here is the base class:

 1public class Mother extends Member implements Visitable {
 2
 3    public Mother(double height, double weight) {
 4        super(height, weight);
 5    }
 6
 7    @Override
 8    public void accept(Visitor visitor) {
 9        visitor.visit(this);
10    }
11}

Now we can define a class responsible to compute family average height:

 1public class AvgHeightVisitor implements Visitor {
 2    private double avgHeight;
 3    private int numMembers;
 4
 5    public AvgHeightVisitor() {
 6        avgHeight = 0;
 7        numMembers = 0;
 8    }
 9
10    @Override
11    public void visit(Father father) {
12        avgHeight += father.getHeight();
13        numMembers++;
14    }
15
16    @Override
17    public void visit(Mother mother) {
18        avgHeight += mother.getHeight();
19        numMembers++;
20    }
21
22    @Override
23    public void visit(Daughter daughter) {
24        avgHeight += daughter.getHeight();
25        numMembers++;
26    }
27
28    @Override
29    public void visit(Son son) {
30        avgHeight += son.getHeight();
31        numMembers++;
32    }
33
34    public double getAvgHeight() {
35        if (numMembers < 1) return 0;
36        return avgHeight / numMembers;
37    }
38}

Let’s run a demo:

 1public class VisitorDemo {
 2
 3    public static void main(String[] args) {
 4        List<Visitable> family = new ArrayList<>();
 5        family.add(new Father(5.6, 150));
 6        family.add(new Mother(5.3, 120));
 7        family.add(new Daughter(2.5, 40));
 8        family.add(new Son(1.5, 20));
 9
10        // compute average height
11        AvgHeightVisitor avgHeightVisitor = new AvgHeightVisitor();
12        for (Visitable member : family) {
13            member.accept(avgHeightVisitor);
14        }
15        System.out.println("Average height = " + avgHeightVisitor.getAvgHeight());
16
17        // compute average weight
18        AvgWeightVisitor avgWeightVisitor = new AvgWeightVisitor();
19        for (Visitable member : family) {
20            member.accept(avgWeightVisitor);
21        }
22        System.out.println("Average weight = " + avgWeightVisitor.getAvgWeight());
23    }
24}
